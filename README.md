# University Database System (SQL Joins)

This project is a Lab Report for the Database Management Systems (CSE210) course. It demonstrates the use of SQL Joins and Group functions in a university database environment.

## About the Project
In this lab, I created a database named university_join_lab. It manages information about:
- Departments: Information about various academic departments.
- Students: Basic student profiles and academic details.
- Courses: Subject titles, credit hours, and department links.
- Enrollments: Records of student marks and semesters.

## Features
I have written SQL queries to:
- Link students with their departments using INNER JOIN.
- Find students who are not yet enrolled in any course using LEFT JOIN.
- Calculate Average, Maximum, and Minimum marks for each course.
- Find the Top 3 Students based on their average marks.
- Filter departments that have more than 2 students using the HAVING clause.

## How to Use
1. Copy the code from the database_script.sql file.
2. Open XAMPP and start the MySQL server.
3. Open phpMyAdmin in your browser.
4. Go to the SQL tab and paste the code.
5. Click Go to execute the script and see the results.
